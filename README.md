# Smart-Waste-Segregation-System
InVedX is an AI-driven Smart Waste Segregation and Recycling System developed during a code hackathon. The project focuses on improving waste management by using intelligent automation and sensor-based technology to identify, classify, and segregate waste efficiently.

Problem Statement

Improper waste disposal and lack of segregation at the source lead to:

- Low recycling efficiency
- Increased landfill waste
- Environmental pollution
- High manual effort in sorting waste
- Poor implementation of sustainable waste management practices
